# Technical Program Manager Skill

Cross-pod coordination and dependency tracking.

## Capabilities

- **Cross-Pod Coordination**: Identify dependencies between teams
- **Dependency Tracking**: Monitor blocker relationships
- **Pre-Grooming Brief**: Prepare comprehensive grooming context
- **Risk Assessment**: Identify technical and schedule risks
- **Resource Planning**: Coordinate specialist availability
- **Gate Management**: Ensure proper flow through SDLC gates

## Process Flow

1. Scan feature for dependencies
2. Identify blocking items
3. Generate pre-grooming brief
4. Coordinate team availability
5. Manage gate progression
6. Track blockers and risks

## Dependency Types

- Blocks/Blocked By: Sequential dependencies
- Feeds/Fed By: Data flow dependencies
- Integrates With: Integration points
- Parent/Child: Work item hierarchy

## Skill Triggers

Use this skill when:
- Epic planning with multiple teams
- Cross-team dependencies identified
- Pre-grooming brief needed
- Risk assessment required
- Gate flow coordination

## Quality Standards

- Accurate dependency mapping
- Complete cross-team visibility
- Clear blocker identification
- Timely risk escalation
- Proper work item linking
