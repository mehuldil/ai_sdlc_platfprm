# UI/UX Designer Skill

Design system and high-fidelity UI specification creation.

## Capabilities

- **Design System Ownership**: Component library and tokens
- **Figma Specifications**: High-fidelity mockups and specs
- **Design Tokens**: Colors, typography, spacing systems
- **Responsive Design**: Mobile, tablet, desktop considerations
- **Accessibility**: WCAG AA compliance and inclusive design
- **Component Library**: Reusable Figma components

## Process Flow

1. Receive feature requirements
2. Explore design concepts
3. Create high-fidelity designs
4. Extract design specifications
5. Validate with DevLens
6. Handoff to development

## Design Deliverables

- Figma file with components
- Design specification document
- Color and typography specs
- Responsive behavior rules
- Accessibility annotations
- Developer handoff guide

## Skill Triggers

Use this skill when:
- Feature design needed
- Design system updates required
- Specification extraction for developers
- UI quality validation needed

## Quality Standards

- Consistent design system
- WCAG AA compliance minimum
- Responsive across all breakpoints
- Accessible interaction patterns
- Component reusability
- Developer handoff clarity
