# Release Management Skill

Release execution and post-launch management.

## Release Responsibilities

- **Release Planning**: Coordinate release schedule
- **Compliance Verification**: Gate approval validation
- **Deployment Execution**: Monitor production deployment
- **Observation Period**: 24-48 hour post-launch monitoring
- **Rollback Management**: Emergency rollback procedures
- **Release Notes**: Generate comprehensive release documentation

## Release Phases

1. **Pre-Release**: Gate approval, readiness verification
2. **Release**: Code deployment, smoke tests
3. **Observation**: Monitoring, metric validation
4. **Post-Release**: Retrospective, documentation

## Compliance Checks

- All gates passed (G1-G9)
- Test coverage validated
- Security audit complete
- Performance NFR met
- Deployment guide reviewed
- Rollback procedure tested
- Monitoring configured

## Skill Triggers

Use this skill when:
- Release approval given
- Deployment coordination needed
- Release notes generation
- Observation period management
- Rollback decision required

## Quality Standards

- Complete gate approval chain
- Accurate release notes
- Clear runbooks
- Proper escalation procedures
- Lesson documentation
