# BOSS Reporting Skill

Executive-level reporting on organizational health and release status.

## Report Types

- **Sprint Health**: Velocity, completion rate, quality metrics
- **Release Readiness**: Feature completion, gate status, risks
- **Dependency Report**: Cross-team blockers and critical path
- **Quality Trends**: Defect patterns and improvements
- **Performance Scorecard**: NFR achievement
- **Risk Dashboard**: Known risks and mitigations

## Metrics Tracked

- Sprint velocity and burn-down
- Story completion rate
- Defect counts by severity
- Cycle time per story
- Test coverage metrics
- Performance NFR status
- Cross-team blockers

## ADO Integration

- Query work items by status
- Aggregate metrics over time
- Track burn-down progression
- Analyze test results
- Monitor gate progression

## Process Flow

1. Collect data from systems
2. Analyze trends and patterns
3. Identify risks and blockers
4. Create executive reports
5. Escalate critical issues
6. Track remediation actions

## Skill Triggers

Use this skill when:
- Executive reporting needed
- Release readiness assessment
- Cross-team health check
- Risk identification needed
- Metrics dashboard update

## Quality Standards

- Accurate data aggregation
- Clear, actionable insights
- Timely escalation
- Cross-team visibility
- Trend analysis
